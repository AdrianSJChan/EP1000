In the previous page I have shared that extrusion is one way to make a 2D sketch 3D, in this page I will be sharing about revolve. Unlike extrusion, the revolve function is for creating 3D objects that are circular or have a proflie that cannot be extruded.

Revolve Function

As you can see above the revolve function needs a center axis to follow, an angle and a face to revolve. For example, with reference to the above image, the profile of revolving is a rectangle profile. This creates a 3D cylinder.
